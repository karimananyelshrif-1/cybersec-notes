
# Proxies

## What is a Proxy?

A proxy is a device that sits between the client and the destination server and acts as a mediator. can inspect or filter traffic.

---

## Types

- Forward Proxy
- Reverse Proxy
- Transparent Proxy
- Non-Transparent Proxy

---

## Forward Proxy

Used to control outgoing traffic.

Example:
- Burp Suite
---

## Reverse Proxy

Protects servers from incoming traffic.

Examples:
- Cloudflare
- Nginx
---

## Transparent Proxy

The client does not know it exists.

---

## Non-Transparent Proxy

The client must configure it manually.

---

## OSI Layer

Layer 7 (Application Layer) -> Burp Suite proxy 
Layer 5 (Session Layer) -> SOCKS Proxy
---

Forward Proxy → Protects Clients

Reverse Proxy → Protects Servers
